# Legal notices and credits

## Conditions for Reusing Garance Content

The RDF data (Linked Open Data) accessible, searchable, and downloadable via this web application was produced by the Archives nationales de France (AnF). This is public information; the user has a non-exclusive and free right to reuse it for commercial or non-commercial purposes worldwide. Each redistribution of the information must include a precise indication of the data's origin ("Archives nationales de France - Garance website for disseminating the RDF version of the reference data ([https://rdf.archives-nationales.culture.gouv.fr/garance](https://rdf.archives-nationales.culture.gouv.fr/garance))") and the date of this data (currently 2026).

If the user wishes to cite the description of an entity (agent, concept, or place) published in Garance, they are asked to use the entity's URI, as displayed on the Garance webpage and as found in the source RDF file.

For example, to cite the description of Simone Veil in Garance, one should write:

“Description of Simone Veil (1927-2017) on the Garance website, which disseminates the RDF version of the reference data of the Archives nationales de France: [https://rdf.archives-nationales.culture.gouv.fr/agent/009941](https://rdf.archives-nationales.culture.gouv.fr/agent/009941)”.

## Data Download

You can download the XML/RDF file for each of the controlled vocabularies, agents, and places from the corresponding webpage in Garance. You can also download a CSV version, generated from the XML/RDF file, of each controlled vocabulary, and the source version, in XML/EAC-CPF format, of most of the agent descriptions.

You can also download all the RDF data from the public repository where we manage it on GitHub: [https://github.com/ArchivesNationalesFR/Referentiels](https://github.com/ArchivesNationalesFR/Referentiels). Please note that **version 2 of this dataset is currently under construction; a consistent release will be published around mid-June 2026**.
