ℹ️(rico:history)

Les premières minutes conservées datent de 1700 : l'étude est installée dans le Marais, d'abord rue d'Orléans, puis rue du Poitou (an IV - an X). Elle déménage alors dans le quartier des affaires : rue du faubourg Montmartre, rue de Provence (1836), rue Lafayette (1866), rue Drouot (1870) et enfin rue Saint-Lazare (1880).
