rico:history

Étude établie rue Saint-Antoine de 1536 à l’an VIII. Elle passe alors dans le faubourg Saint-Germain, s’établit sur les quais, puis rue du Bac (1817). En 1847, elle est rue de l’Université. En 1852, elle se fixe dans le quartier Montmartre : rue du Faubourg, rue Notre-Dame-de-Lorette (1861)
